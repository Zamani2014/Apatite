Αμεση Βοήθεια (Έκδοση 1.0)
Greek Translation: Ioannis Goutsidis
Καλωσήρθατε στο Microsoft Homepage Starter Kit, ενός εύκολου στην χρήση ASP.NET 2.0 Συστήματος Διαχείρισης Περιεχομένου (CMS), το οποίο επιτρέπει τη δημιουργία προσωπικών δυναμικών ιστοχώρων (Websites). Ο παρόν οδηγός, παρέχει μια γρήγορη επισκόπηση στη διαχείριση και τις δυνατότητες της εφαρμογής.

Εάν αναζητάτε κάτι συγκεκριμένο, χρησιμοποιήστε τη "Find" επιλογή του browser σας. Το παρόν  CMS διαθέτει πληθώρα δυνατοτήτων που περιμένουν να τις ανακαλύψετε.

Για επιπρόσθετες πληροφορίες, επισκεφθείτε το My Web Pages Starter Kit project στο δικτυακό τόπο του Codeplex.com.

Διαχείριση:
1. Εγκατάσταση ιστοχώρου (Website)

Πριν τη δημοσίευση περιεχομένου, είναι απαρίτητη η αρχική ρύθμιση κάποιων γενικών παραμέτρων του νέου σας ιστοχώρου. Μετά την πρώτη σας σύνδεση, μεταβείτε στο μενού Διαχείριση και επιλέξτε Ρυθμίσεις CMS συστήματος:
Από την παρακάτω σελίδα, μπορείτε να αλλάξετε κάποιες από τις γενικές ρυθμίσεις, καθώς επίσης και τη συνολική εικόνα (layout) του ιστοχώρου σας, ακόμη και μετά την εισαγωγή του όποιου περιεχομένου!
Περιγραφή όρων:

Τίτλος
Ο τίτλος αποτελεί το όνομα του ιστοχώρου σας και εμφανίζεται στην αρχική εικόνα αυτού ως λογότυπο.

Γλώσσα
Οι μελλοντικές εκδόσεις του προϊόντος, θα είναι διαθέσιμες σε διάφορες γλώσσες.

SMTP Server
Εάν επιθυμείτε την χρήση δυνατοτήτων, οι οποίες απαιτούν την αποστολή e-mails, πρέπει να ορίσετε τον e-mail διακομιστή σας.

Υποσέλιδο
Εδώ εισάγονται πληροφορίες επικοινωνίας ή πνευματικών δικαιωμάτων, οι οποίες παρουσιάζονται στο τέλος κάθε σελίδας.

Περιγραφή ιστοχώρου & Λέξεις-κλειδιά ιστοχώρου
Εδώ εισάγετε τις επιπλέον πληροφορίες (meta-data)για τις κεφαλίδες (headers) του ιστοχώρου σας.

Αρχικές ρυθμίσεις
Επιστροφή του ιστοχώρου στην αρχική του κατάσταση.
ΠΡΟΣΟΧΗ: Διαγράφονται όλα τα δεδομένα!

Ενεργοποίηση RSS τμήματος
Πολλά από τα modules έχουν δυνατότητες RSS feeds. Με την ενεργοποίηση του RSS είναι δυνατή η εμφάνιση του αντίστοιχου εικονιδίου στο τέλος των τμημάτων. Οι επισκέπτες μπορούν να εγγραφούν στις ενημερώσεις του συγκεκριμένου τμήματος πατώντας στο εικονίδιο.

Επιλογή θέματος
Μια πολύ ωραία δυνατότητα του CMS αυτού είναι το γεγονός ότι μπορείτε ανά πάσα στιγμή να αλλάξετε τη μορφή του με ένα και μόνο κλικ. Όλο το περιεχόμενο αναπροσαρμόζεται αυτομάτως και γίνεται χρήση νέων style sheets.

Απλά επιλέγετε ένα νέο θέμα και πατάτε επάνω του. Είστε έτοιμοι!

Εάν είστε εξοικειωμένοι/ες με το .Net και το Visual Studio, μπορείτε να δημιουργήσετε ακόμη και τα δικά σας θέματα!
2. Διαχείριση σελίδων & πλοήγηση

Από την επιλογή Σελίδες & Πλοήγηση έχετε τον έλεγχο της δομής του ιστοχώρου σας, οργανώνετε την πλοήγηση και δημιουργείτε νέες σελίδες.

Οι σελίδες παρουσιάζονται με τη σειρά που εμφάνιζονται κατά την πλοήγηση.

Οι εσωτερικές σελίδες (π.χ. "...Εσωτερική σελίδα" της εικόνας), αποτελούν δευτερεύουσες σελίδες της αμέσως προηγούμενης και κατά την πλοήγηση εμφανίζονται στο αντίστοιχο υπο-μενού.

Για την προσθήκη μιας νέας σελίδας αρκεί ένα πάτημα στο κουμπί "Νέα σελίδα". Μια νέα σελίδα εμφανίζεται στο τέλος της λίστας.

Για τη μετακίνηση μιας σελίδας στην ιεραρχία της πλοήγησης ή για να γίνει μια σελίδα εσωτερική κάποιας άλλης, απαιτείται η επιλογή της και κατόπιν η χρήση των βελών μετακίνησης. Το δεξιό βέλος ορίζει μια σελίδα ως εσωτερική, ενώ τα επάνω και κάτω βέλη αλλάζουν τη θέση μιας σελίδας εντός της ιεραρχίας.
Περιγραφή όρων:

Τίτλος σελίδας
Ο τίτλος που εμφανίζεται στο παράθυρο του browser.

Πλοήγηση
Το όνομα της σελίδας που εμφανίζεται στο μενού πλοήγησης.

Επιτρέπονται ανώνυμοι χρήστες
Εξ' ορισμού όλοι οι επισκέπτες έχουν πρόσβαση σε όλες τις σελίδες. Στην περίπτωση που επιθυμείτε να διατηρείτε μια περιοχή μόνο για μέλη, μπορείτε να απο-επιλέξετε αυτή την επιλογή και έτσι, μόνο εγγεγραμμένοι χρήστες θα έχουν πρόσβαση εκεί. (-> δείτε 3. Διαχείριση χρηστώ )

Editing by power users
If you activate this button, power users can edit this page. This enables the delegation of administrative areas (-> δείτε 3. Διαχείριση χρηστώ )

Ορατό
Για την απόκρυψη μιας σελίδας από το μενού πλοήγησης (λόγω π.χ. εργασιών), απο-επιλέξτε το στοιχείο αυτό.
Εικονική διαδρομή:
Στα πλαίσια ενός δυναμικού CMS οι νέες σελίδες φέρουν γενικά IDs. Για να είναι τα URLs των σελίδων αυτών πιο φιλικά προς τους χρήστες, μπορείτε να τους αναθέσετε εικονικά ονόματα διαδρομής.
Απλά, εισάγετε ένα όνομα για τη σελίδα, π.χ. "Βιογραφικό". Αντί λοιπόν να έχετε γενικά URLs όπως www.τοsiteσας.com/default.aspx?pg=3cb13823-a424-4761-81c1-5fa9da24db43, μπορείτε να χρησιμοποιείτε μια πιο μικρή έκδοση του τύπου "www.τοsiteσας.com/bio.aspx"

Με τον τρόπο αυτό, η παρουσίαση ενός URL στα πλαίσια εμπορικού υλικού ή αποστολής του με  e-mail γίνεται πολύ πιο εύκολη!
3. Διαχείριση χρηστών

Το CMS παρέχει ένα απλό στην χρήση αλλά με πολλές δυνατότητες εργαλείο διαχείρισης χρηστών, το οποίο μπορεί να χρησιμοποιηθεί για τη δημιουργία προστατευμένων περιοχών.
Έχετε υπόψη σας, ότι υπάρχουν 2 επίπεδα δικαιωμάτων:

1. Εγγεγραμμένοι χρήστες: Σελίδες μη προσβάσιμες από τους πάντες (δείτε 2. Διαχείριση σελίδων & πλοήγηση), στις οποίες έχουν πρόσβαση μόνον εγγεγραμμένοι χρήστες. Υπάρχει ένα μόνο επίπεδο ασφάλειας στους εγγεγραμμένους χρήστες.

2. Διαχειριστές: Εκτός των εγγεγραμμένων χρηστών, μπορεί να υπάρχουν και ένας ή/και περισσότεροι διαχειριστές. Οι διαχειριστές έχουν πλήρη πρόσβαση σε όλες τις δυνατότητες του site, μπορούν να το αναμορφώσουν, να δημιουργήσουν νέες σελίδες, να διαγράψουν υπάρχουσες και να διαχειριστούν τους χρήστες.

3. Power-User: Registered Users can be defined as Power Users. Power Users have unrestricted access to pages, for which have been given access set by Administrator. The Backend is only accessible for Administrators.
Για τη δημιουργία νέων χρηστών, αρκεί η εισαγωγή ενός ονόματος, ενός κωδικού πρόσβασης και μιας e-mail διεύθυνσης. Τέλος πατάτε στο "Δημιουργία χρήστη".
Πατώντας στο κουμπί Επεξεργασία, είναι δυνατή η αλλαγή του ονόματος και της e-mail διεύθυνσης ενός χρήστη. Μπορείτε να αναθέστε δικαιώματα διαχειριστή σε έναν χρήστη ενεργοποιώντας την επιλογή "Διαχειριστής".

Η τελευταία σύνδεση του εκάστοτε χρήστη παρουσιάζεται εδώ.
Δυνατότητες (Modules):
Κάθε νέα σελίδα είναι κενή και εμφανίζει το στοιχείο ελέγχου για την προσθήκη κάποιου τμήματος:
Αυτή τη στιγμή είναι διαθέσιμα 10 στοιχεία ελέγχου (modules) για τις σελίδες σας. Μπορείτε να προσθέσετε όσα στοιχεία ελέγχου επιθυμείτε και να καθορίσετε τη σειρά εμφάνισης αυτών. Στο τέλος κάθε σελίδας υπάρχει πάντοτε η δυνατότητα προσθήκης ενός νέου στοιχείου ελέγχου.

Για την αλλαγή της σειράς εμφάνισης των στοιχείων ελέγχου, αρκεί η αντίστοιχη επιλογή των βελών που υπάρχουν στην κεφαλίδα κάθε module, τα οποία το μεταφέρουν επάνω ή κατώ κατά μια θέση κάθε φορά.
Το κουμπί Διαγραφή τμήματος διαγράφει πλήρως το αντίστοιχο τμήμα.

Πολλά από τα modules φέρουν έναν κειμενογράφο (rich text editor) για τη δημιουργία περιεχομένου. Αυτός επιτρέπει τη μορφοποίηση του κειμένου, τη δημιουργία πινάκων κ.λπ. - μπορείτε να ανακαλύψετε μόνοι σας όλα τα εργαλεία που παρέχονται...
Για την προσθήκη μιας εικόνας, αρκεί ένα πάτημα στο αντίστοιχο κουμπί και είτε επιλέγετε την εικόνα από κάποιο διακομιστή (server), ή χρησιμοποιείτε το εργαλείο ανεβάσματος (upload). Εάν είστε εξοικειωμένοι/ες με την HTML, μπορείτε να μεταβείτε στον αντίστοιχο HTML κώδικα, από την σχετική επιλογή.


Προβολή και Επεξεργασία

Για να μπορούν οι διαχειριστές να έχουν μια πραγματική εικόνα της εργασίας τους, χωρίς να απαιτείται η αποσύνδεσή τους, τα modules μπορούν να λειτουργήσουν σε δύο καταστάσεις: Προβολή και Επεξεργασία. Η μετάβαση σε καθεμιά από αυτές τις καταστάσεις λειτουργίας, πραγματοποιείται μέσω των αντίστοιχων κουμπιών.
Αποστολή στην μπάρα
Κάποια στοιχεία ελέγχου παρέχουν την επιλογή "Αποστολή στην μπάρα".
Με την ενεργοποίηση της επιλογής αυτής, το περιεχόμενο των modules εμφανίζεται σε μια πλευρική μπάρα του ιστοχώρου, η οποία μπορεί να χρησιμοποιηθεί ως τμήμα γρήγορων συνδέσμων ή τελευταίων ειδήσεων.
1. Φόρμα επικοινωνίας

Μείνετε σε επαφή με φίλους, συναδέλφους και πελάτες παρέχοντας έναν εύκολο τρόπο αποστολής μηνυμάτων προς εσάς: η Φόρμα επικοινωνίας. Έχετε υπόψη ότι απαιτείται ο καθορισμός ενός SMTP server στις γεινκές ρυθμίσεις του ιστοχώρου.
Περιγραφή όρων:

Οι ρυθμίσεις της Φόρμας επικοινωνίας έχουν τρεις στόχους: Τη δημιουργία μιας φόρμας επικοινωνίας, τον καθορισμό του παραλήπτη των μηνυμάτων και την παροχή ενός ευχαριστήριου μηνύματος προς τον επισκέπτη.

Τα τρία πρώτα πεδία Ε-mail παραλήπτη, Αντιγραφή σε και Θέμα καθορίζουν τον τρόπο, με τον οποίο παραδίδονται σε εσάς τα e-mails.

Το Κείμενο εισαγωγής αποτελεί την πρόσκλησή σας στους επισκέπτες να επισκοινωνήσουν μαζί σας.

Το Μήνυμα ευχαριστίας εμφανίζεται στην οθόνη του επισκέπτη, αφού ο τελευταίος αποστείλει το μήνυμα, μέσω της φόρμας επικοινωνίας.
2. Downloads

Για το διαμοιρασμό εγγράφων από τον ιστοχώρο σας χρησιμοποιήστε το module Downloads. Σας επιτρέπει να ανεβάσετε διάφορα αρχεία, για τα οποία δημιουργείται αυτόματα το αντίστοιχο εικονίδιο εφαρμογής.
3. Easy Control

Επιθυμείτε την ενσωμάτωση της δικής σας δυνατότητας (UserControl) στο CMS; Το πιο εύκολο: το Easy Control αποτελεί το τμήμα που πρέπει να επιλέξετε! Απλά δημιουργείτε το δικό σας στοιχείο ελέγχου (UserControl) και το τοποθετείτε στον κατάλογο EasyControls του ιστοχώρου σας. Μπορείτε ακόμη και να αντιγράψετε υπάρχοντα στοιχεία ελέγχου. Με την τοποθέτηση ενός  Easy Control τμήματος σε κάποια από τις σελίδες σας, όλα τα στοιχεία ελέγχου εντός του καταλόγου EasyControls εμφάνιζονται αυτόματα στη λίστα επιλογής, η οποία εμφανίζεται στην κατάσταση επεξεργασίας του τμήματος. Απλά επιλέγετε το επιθυμητό στοιχείο ελέγχου που θέλετε να εμφανίσετε και είστε έτοιμοι/ες. Εύκολο έτσι;
4. Γεγονότα

Προσκαλέστε κάποιος επισκέπτες και μεταφέρετε τα γεγονότα σας online. Το Event List module σας επιτρέπει να δημιουργήσετε γεγονότα και να ορίσετε ημερομηνίες και ώρες.
Περιγραφή όρων:

Δημιουργία γεγονότος:

Επιλέγετε την ημερομηνία του γεγονότος με το αντίστοιχο δυναμικό εργαλείο.

Επιλέγετε την ώρα και τα λεπτά από τις αντίστοιχες λίστες επιλογής.

Το πεδίο "Εμφάνιση από" είναι όπως την ημερομηνία δημοσίευσης. Εάν επιλέξετε κάποια ημερομηνία, το γεγονός θα είναι ορατό μόνο μετά από αυτή την ημερομηνία.
5. Φωτογραφίες

Για το διαμοιρασμό των φωτογραφιών σας χρησιμοποιείτε το module Φωτογραφίες. Μπορούν να επιλεγούν αρχεία με την αντίστοιχη σήμανση όπως και στο module Downloads.
Με τη δημοσίευση των φωτογραφιών δημιουργούνται αυτόματα οι αντίστοιχες μικρογραφίες.
Δημιουργείτε επίσης μια φωτογραφία επισκόπησης, η οποία ταιριάζει με τις διαστάσεις του  module. Η αρχική φωτογραφία είναι διαθέσιμη μέσω συνδέσμου για download.
6. Βιβλίο επισκεπτών

Επιτρέπει στους επισκέπτες σας να κάνουν τα σχόλιά τους για τον ιστοχώρο. Το βιβλίο επισκεπτών είναι εύκολο στη ρύθμιση και συνοδεύται από εικόνα ασφαλείας για μεγαλύτερη προστασία.
Στην έκδοση 1 δεν μπορείτε να αλλάξετε τις καταχωρήσεις στο επίπεδο παρουσίασης του ιστοχώρου.
7. HTML περιεχόμενο

Το πλέον σημαντικό τμήμα - παρέχει τη μεγαλύτερη ευελιξία. Με τον κειμενογράφο είστε σε θέση να αλλάξετε τη μορφοποίηση του περιεχομένου και να επεκτείνετε τις δυνατότητες.

Για την προσθήκη μιας εικόνας, αρκεί ένα πάτημα στο αντίστοιχο κουμπί και είτε επιλέγετε την εικόνα από κάποιο διακομιστή (server), ή χρησιμοποιείτε το εργαλείο ανεβάσματος (upload). Εάν είστε εξοικειωμένοι/ες με την HTML, μπορείτε να μεταβείτε στον αντίστοιχο HTML κώδικα, από την σχετική επιλογή.
8. Σύνδεσμοι

Το module Σύνδεσμοι λειτουργεί όπως και τα Downloads/Φωτογραφίες. Προστίθεται το URL και ο στόχος επιλέγεται ξεχωριστά.
9. Ειδήσεις

Προσθήκη ειδήσεων στον ιστοχώρο σας. Επιλέγετε τις ημερομηνίες "Εμφάνιση από" και "Εμφάνιση έως" για να καθορίσετε τη διάρκεια εμφάνισης των ειδήσεων στον ιστοχώρο σας.
10. Υπο-σελίδες

Στην περίπτωση που διαθέτετε μια σελίδα με επιπλέον δευτερεύουσες σελίδες, ίσως είναι προτιμότερο να έχετε μια λίστα συνδέσμων, η οποία να παραπέμπει στις διάφορες σελίδες, όπως ένας πίνακας περιεχομένων για μια κατηγορία του ιστοχώρου σας.

Το module Υπο-σελίδες εκτελεί αυτήν ακριβώς τη λειτουργία: Απλά το προσθέτετε στην κύρια σελίδα σας και δημιουργείται αυτόματα μια λίστα με συνδέσμους για όλες τις δευτερεύουσες σελίδες!
11. Blog

The Blog Module provides you with all the advantages of a Web log:

  • Publicity for your Website: the ping function attracts more visitors to your Website. 
  • Topicality of a selected thread: keep the visitors posted (travel diary, progress of a project, monitoring of publications in the Internet, knowledge management, etc.).
  • Communication and exchange of ideas: read the comments of your blog visitors and reply to them by updating your blog entry or posting new entries.

Post new blog entries or edit old ones.

Tags:
Click on Tags and a little window opens where you can assign tags to an entry or create new ones.



Keywords can be changed or erased anytime. Just click on Tags in the Edit mode of the blog.



Weblogs input notification:
You can decide in the configuration of the blog whether a ping should be sent to Weblogs.com when posting entries. This will advertise your input with Weblogs.



Calendar:
The calendar displays the days on which the entries were posted in bold.



Comments:
Visitors can post comments on any blog entry which are immediately published. An administrator is able to erase these in the Edit mode.



12. Blogroll

The Blogroll module has a similar structure to the Link List. The only difference is that the Blogroll displays links to other blogs you would like to introduce to your visitors.



To post a new entry, please indicate a title and the URL of the post. You can also add the RSS feed to the post. This enables the user to subscribe to all blogs with one click (Subscription List).
13. TagCloud

The purpose of the TagCloud is to visualize keywords used in the blog. Keywords used more often are displayed in bold. Clicking on a keyword displays all entries containing this keyword.



You can place the TagCloud as often as you like and anywhere. If you want your Website to feature more than one blog, the TagCloud groups all blogs together
